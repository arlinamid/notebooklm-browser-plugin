# Forrásalapú Projekttervező

**Forrás:** Excellent AI Prompts
**Nehézség:** Középhaladó

## Prompt

```
Te egy tanulástervező szakértő vagy, aki a feltöltött anyagok bizonyítékaira alapozva tervez projekteket.

Hivatkozz keresztbe az összes feltöltött forrásom között, és tervezz meg egyetlen fókuszált projektet:

Célom: [SPECIFIC SKILL I WANT TO DEVELOP]
Rendelkezésre álló idő: [HOURS PER WEEK]

A forrásaimból vett idézetekkel add meg a következőket:
- EGY projekt, amely több forrásban is szerepel (idézd, melyik források emlegetnek hasonló megközelítéseket)
- A pontos módszertan az A forrás és a B forrás és a C forrás leírása szerint
- A különböző forrásokban említett konkrét eszközök/technikák (oldalszámmal hivatkozva)
- A sikerességi benchmarkok, amelyeket az anyagok szó szerint tartalmaznak
- A források közötti ellentmondások és azok feloldásának módja

Formátum: "A [Forrás neve, X. oldal] szerint: '[pontos idézet]', míg a [Y forrás] azt állítja: '[ellentétes idézet]'"

Összpontosíts arra, amit a források valóban javasolnak, ne általános projektötletekre.
```

## Felhasználási eset (adatmérnökség)

Töltsd fel a DLT dokumentációt + FastAPI oktatóanyagokat + PostgreSQL útmutatókat → kérj egy konkrét adatpipeline-projektet, amely mindhárom technológiát integrálja.
