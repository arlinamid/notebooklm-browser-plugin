# Egyéni hangáttekintő vitával

**Forrás:** XDA Developers
**Nehézség:** Haladó

## Prompt

```
Készíts egy hangáttekintőt, amelyben a két műsorvezető vitatkozik, és alkalmanként eltérő véleményen vannak az értelmezések kapcsán.

Amikor nem értenek egyet, tegyék a következőket:
1. Fogalmazzák meg egyértelműen a nézeteltérést
2. Hivatkozzanak az egyes álláspontokat alátámasztó konkrét forrásokra
3. Oldjanak fel a vitát a helyes értelmezés részletes magyarázatával

A vita formátumán keresztül tegyék emlékezetessé az alapfogalmakat.
```

## Felhasználási eset (adatmérnökség)

Tölts fel egymásnak ellentmondó dokumentumokat (pl. SQL vs NoSQL az analitikában) → strukturált vita hangáttekintője feloldással.
