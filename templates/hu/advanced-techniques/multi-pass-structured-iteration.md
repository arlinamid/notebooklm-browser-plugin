# Többlépéses strukturált iteráció

**Forrás:** Matt Wolfe
**Nehézség:** Haladó

## Prompt

```
Dolgozd fel ezt 4 lépésben:

1. lépés: Foglald össze az összes forrás főbb érveit
2. lépés: Azonosítsd az ellentmondásokat vagy nézeteltéréseket
3. lépés: Javasolj szintézist, amely feloldja az ellentmondásokat
4. lépés: Készíts megvalósítható cselekvési tervet

Minden lépést fejezz be, mielőtt a következőre lépsz. Mutasd be az egyes lépések munkamenetét.
```

## Felhasználási eset (adatmérnökség)

Összetett architekturális döntések, amelyek többforrású szintézist igényelnek.
