# Többforrású összehasonlító táblázat

**Forrás:** Matt Wolfe
**Nehézség:** Középhaladó

## Prompt

```
Hasonlítsd össze az alábbi 3 kategóriát az összes feltöltött forrásban:
1. Ajánlott módszertanok (előfordulási gyakorisággal)
2. Említett eszközök/keretrendszerek (megemlítések száma szerint rangsorolva)
3. Meghatározott sikermetrikák (pontos idézetek kiemelésével)

Jelenítsd meg összehasonlító táblázatban, forráshivatkozásokkal.
```

## Felhasználási eset (adatmérnökség)

Ha 20+ forrásod van, és azonosítani szeretnéd a konszenzust és az eltéréseket.
