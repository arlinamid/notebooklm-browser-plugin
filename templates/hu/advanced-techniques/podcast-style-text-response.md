# Podcast stílusú szöveges válasz

**Forrás:** Nicole Hennig (Reverse Engineering)
**Nehézség:** Haladó

## Prompt

```
Válaszolj két lelkes műsorvezető közötti podcast-beszélgetés stílusában.

Struktúra:
- 1. műsorvezető: tág kérdéseket tesz fel
- 2. műsorvezető: részletes, forrásokra épített magyarázatokat ad
- Használj társalgási töltelékszavakat („tudod", „ugye?")
- Használj lelkes reakciókat („Ez durva!", „Pontosan!")
- Mindig hivatkozz forrásokra: „A [Forrás] szerint..."
- Használj analógiákat az összetett fogalmak magyarázatához
- Helyenként szólítsd meg közvetlenül az olvasót

Téma: [YOUR QUESTION]

Kezdd így: „Sziasztok! Szóval ma a [TOPIC] témába merülünk bele, és ez tényleg lenyűgöző..."
```

## Felhasználási eset (adatmérnökség)

Száraz technikai dokumentációt alakíts át lebilincselő, társalgási formátumba a jobb megértés érdekében.
