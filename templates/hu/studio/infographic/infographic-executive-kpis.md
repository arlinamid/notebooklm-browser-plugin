# Infografika — Vezetői/KPI-k

**Forrás:** Custom - Validated
**Tájolás:** Square (1:1)
**Részletesség:** Concise

## Irányítóprompt

```
Svájci minimalizmus stílusú infografika, szigorú rácsigazítás, nagy kontrasztú elrendezés.
Tengerészkék alap arany kiemelőkkel a kulcsmutatókhoz, törtfehér háttér.
Kiemelten megjelenített fő statisztika kiegészítő adatpontokkal.
Letisztult talp nélküli betűtípus, professzionális vezetői esztétika.
Minimális díszítés, az adatok magukért beszélnek.
```

## Dashboard változat

```
Vezetői dashboard stílus, letisztult adatvizualizációs esztétika.
Sötét tengerészkék háttér élénk kiemelőszínekkel a mutatóknál.
Kulcs KPI-k nagy számként, trendjelzőkkel.
Strukturált negyedek vagy szekciók a különböző mutató-kategóriákhoz.
Professzionális vállalati megjelenés, 10 másodperc alatt áttekinthető.
```

## Felhasználási eset (adatmérnökség)

KPI-k és mutatók vizualizációja vezető stakeholderek számára. Ideális a következőkhöz:
- Negyedéves riportok kiemelő statisztikákkal
- Pipeline-teljesítmény dashboardok
- Pénzügyi összefoglalók és kulcsmutatók

**Validálva tesztelve:** Meta Q4 2024 Earnings Presentation
