# Infografika — Közösségi média

**Forrás:** Custom - Validated
**Tájolás:** Square (1:1)
**Részletesség:** Concise

## Irányítóprompt (Cyberpunk — Ajánlott tech/AI tartalomhoz)

```
Tech trend infografika, cyberpunk-inspirált sötét téma.
Sötét háttér neon türkizkék és magenta kiemelőkkel.
Egy erős statisztika vagy előrejelzés kiemelten megjelenítve.
Ragyogó effektusok a kulcselemeken, futurisztikus talp nélküli betűtípus.
Azonnali megértésre tervezve, közösségi médiára optimalizálva.
```

## Gradiens változat (Vállalati/Semleges)

```
Figyelemfelkeltő infografika, szemet ragadó modern esztétika.
Élénk gradiens háttér, mélylilától elektromos kékig átmenve.
Egyetlen kiemelő statisztika vagy felismerés fókuszpontként, nagy félkövér tipográfiával.
Minimális kiegészítő szöveg, nagy vizuális hatás a görgetés megállításához.
Letisztult ikonográfia, zsúfoltság nélkül.
```

## Felhasználási eset (adatmérnökség)

Virális közösségi médiás tartalom létrehozása. Ideális a következőkhöz:
- Feltűnő AI/ML statisztikák
- Megosztható tech trendek
- Adatelemzések LinkedIn/Twitter számára

**Validálva tesztelve:** Our World in Data - AI

**Megjegyzés:** A Cyberpunk változat jobb kontrasztot és vizuális hatást nyújt, mint a Gradiens változat tech tartalmakhoz.
