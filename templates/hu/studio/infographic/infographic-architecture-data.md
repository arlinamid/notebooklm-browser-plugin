# Infografika — Architektúra/Adat

**Forrás:** Custom - Validated
**Tájolás:** Landscape (16:9)
**Részletesség:** Standard

## Irányítóprompt

```
Izometrikus műszaki diagram, letisztult tervrajz-esztétika, lágy kék és szürke paletta türkizkék kiemelőkkel.
Az összetevőket összekötött csomópontokként jelenítsd meg iránnyal jelölt adatfolyam-nyilakkal.
Minimális szöveges felirat, az ikonokra és a vizuális hierarchiára támaszkodj.
Balról jobbra haladó folyamatként strukturálva: forrás → feldolgozás → tárolás → felhasználás.
```

## Tömör változat

```
Izometrikus diagram, tervrajz-stílus, kék-szürke duotón.
Az összetevők csomópontként, nyilak jelzik az adatfolyam irányát.
Minimális szöveg, csak ikonok.
```

## Felhasználási eset (adatmérnökség)

Adatplatform-architektúra vizualizációja technikai dokumentációhoz. Ideális a következőkhöz:
- Medallion Architecture diagramok (Bronze/Silver/Gold)
- ETL/ELT pipeline-ok iránnyal jelölt folyamatokkal
- Felhőarchitektúrák (Azure Databricks, Snowflake stb.)

**Validálva tesztelve:** Microsoft Learn - Azure Databricks Architecture
