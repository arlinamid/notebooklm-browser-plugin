# Infografika — Ismeretterjesztés/Fogalmak

**Forrás:** Custom - Validated
**Tájolás:** Portrait (9:16)
**Részletesség:** Concise

## Irányítóprompt

```
Flat design illusztráció, Kurzgesagt-stílus, lágy pasztellszínek meleg kiemelőkkel.
Egyszerűsített lekerekített formák, barátságos ikonok, letisztult vizuális folyamat.
Minimális szöveg, számozott lépésekre és vizuális metaforákra támaszkodj.
Felülről lefelé haladó struktúra: fogalom → mechanizmus → eredmény.
```

## Mesekönyv változat

```
Oktatási infografika, gyermekmesekönyv-esztétika, lágy pasztell paletta.
Lekerekített formák, egyszerű ikonok, vizuális történetmesélő megközelítés.
Számozott szekciók függőlegesen haladva, szekciónként egy kulcsgondolattal.
Csak minimális feliratok, az illusztrációk hordozzák a magyarázatot.
```

## Felhasználási eset (adatmérnökség)

Technikai fogalmak magyarázata nem szakembereknek. Ideális a következőkhöz:
- Bevezetés a Machine Learningbe stakeholdereknek
- Felhőfogalmak ismeretterjesztése (serverless, containers)
- Nem technikai onboarding adatpipeline-okról

**Validálva tesztelve:** Ntiva - Machine Learning Beginner's Guide
