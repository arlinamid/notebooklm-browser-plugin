# Infografika — Folyamat/Idővonal

**Forrás:** Custom - Validated
**Tájolás:** Portrait (9:16)
**Részletesség:** Standard

## Irányítóprompt

```
Izometrikus utazásillusztráció, felülről lefelé kanyargó ösvénnyel.
Lágy kék és türkizkék paletta, meleg kiemelőszínű mérföldkő-jelzőkkel.
Minden lépés önálló állomás az ösvényen, számozott ikonokkal.
Letisztult összekötő vonalak a haladás és a függőségek megjelenítéséhez.
Minimális szöveges felirat, vizuális történetmesélés ikonokkal és térbeli elrendezéssel.
```

## Klasszikus idővonal változat

```
Függőleges idővonal infografika, letisztult modern esztétika.
Középső gerinc felváltva bal és jobb oldali tartalomblokkal.
Számozott mérföldkövek, fázisanként eltérő ikonokkal.
Lágy gradiens háttér, professzionális színpaletta.
Egyértelmű vizuális hierarchia: fázisszám → cím → rövid leírás.
```
