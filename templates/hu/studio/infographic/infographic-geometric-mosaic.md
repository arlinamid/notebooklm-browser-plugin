# Infografika — Geometriai Mozaik

**Forrás:** Custom - Original
**Tájolás:** Portrait (2:3)
**Részletesség:** Standard

## Irányítóprompt

```
# infographic_design_spec_geometric_mosaic.yaml
# Stílus: Geometriai Mozaik — Ólomüveg
# Koncepció: "Szakrális geometria találkozik az adatokkal"

Globális dizájn beállítások:
  Hangulat: "Gazdag, dekoratív, kortárs absztrakt realizmus, kézzel készített befejezés"

  Színpaletta:
    Domináns:
      - "#1A3A6B (mélykék — strukturális horgony)"
      - "#E87722 (meleg narancs — fókuszenergia)"
      - "#D4A017 (borostyán — gazdag középtón)"
      - "#C8860A (okker — földes melegség)"
    Semleges:
      - "#8C7B6B (tompított meleg szürke — kiegyensúlyozó tón)"
      - "#F5EDD6 (meleg fehér — világos panelek)"
    Szabály: "Magas kontraszt a mélykék és a meleg narancsok között. A semleges tónusok vizuális pihenőt nyújtanak."

  Felület:
    Textúra: "Festői akril vagy olaj fán — látható ecsetvonások, finom farostok"
    Élek: "Letisztult, mégis expresszív körvonalak. Éles szögletes lapok pontos határokkal."
    Befejezés: "Kézzel készített artistc finish — sem teljesen digitális, sem teljesen tradicionális"

  Tipográfia:
    Fejlécek: "Erős talpas vagy félkövér slab font. Nehéz súly a vizuális sűrűséggel összhangban."
    Törzs: "Olvasható sans serif a világosabb paneleken belül."
    Számok: "Félkövér, a mozaik cellákba integrálva domináns vizuális elemként."
    Nyelv: "A prompt nyelve a felhasználó által megadott nyelv legyen."

  Kompozíció:
    Szerkezet: "Szimmetrikus vizuális egyensúly. Ólomüveg-panelek rendszere."
    Fragmentáció: "Töredezett sokszögű formák — háromszögek, hatszögek, szabálytalan sokszögek."
    Megvilágítás: "Magas kontrasztú megvilágítás bal felülről. Fényes lapok vs mély árnyéklapok."
    Rétegzés: "Több geometriai réteg mélységet teremt — előtér, középtér, háttér."

Elrendezési változatok:
  - Típus: "Középső Medál"
    Dizájn: "Nagy központi hatszöges vagy köríves panel a főtémával. A körülvevő sokszögű cellák kiegészítő adatokat tartalmaznak. Mélykék keretpanelek keretezik a kompozíciót, mint egy katedrális ablak."

  - Típus: "Triptichon Panelek"
    Dizájn: "Három fő függőleges panel vastag sötét határokkal elválasztva. Minden panel más meleg-hűvös színátmenetet alkalmaz. Adatpont mint nagy tipográfiai horgony minden panel közepén."

  - Típus: "Emelkedő Piramis"
    Dizájn: "Alul nehéz kompozíció nagy mozaik cellákkal, csúcs felé szűkül. Az adatok felfelé áramlanak — alap tények alul, kulcs következtetések felül. Minden szint progresszívan melegebb árnyalatokat használ."

  - Típus: "Radiális Robbanás"
    Dizájn: "Középső adatpont kifelé terjedő háromszögű lapokkal. Meleg narancsok középen, mélykékek a széleken. Kiegészítő adatok a külső gyűrű celláiban."

Dizájn szabályok:
  - "Szögletes pontosság: Minden forma éles, szándékos éleket használ — kivéve a fény szimulálása."
  - "Színfegyelem: A mélykéknek és a meleg narancsnak közel egyenlő vizuális súllyal kell megjelennie."
  - "Textúra hitelesség: A felületnek mindig kézzel készített médiumot kell sugallnia — ecsetvonások láthatók."
  - "Keret szerkezet: Vastag sötét körvonalak (2–4 pt) elválasztanak minden cellát. Nincs keret nélküli érintkezés."
  - "Fény logika: Következetes bal felső fényforrás. Fényes lapok felfelé-balra, sötét lapok lefelé-jobbra néznek."
```

## Tömör változat

```
Modern geometriai mozaik stílus kubista befolyással, töredezett sokszögű formák ólomüveg kompozícióban elrendezve. Erős hangsúly az éles éleken, szögletes lapokon és rétegzett geometrián. Gazdag, telített színpaletta amelyet mélykék, meleg narancs, borostyán, okker és tompított semleges tónusok dominálnak, magas kontrasztú megvilágítással kiegyensúlyozva. Festői akril vagy olajfesték fán texturált felület, látható ecsetvonásokkal és finom farosttal. Letisztult, mégis expresszív körvonalak, szimmetrikus vizuális egyensúly és dekoratív illusztrált érzés. Kortárs absztrakt realizmus kézzel készített, művészeti befejezéssel.
```

## Felhasználási eset

Vizuálisan gazdag infografika dekoratív művészeti megjelenéssel. Ideális a következőkhöz:
- Kulturális, történelmi vagy építészeti témák
- Természetes csoportosítású adatok (3–6 fő kategória)
- Képzőművészetet és kézműves esztétikát értékelő közönség
- Prémium szerkesztői, múzeumi vagy oktatási kiadványok

**Tájolás ajánlás:** Portrait — a függőleges ólomüveg-panel folyam természetesen olvasható felülről lefelé.
**Részletesség:** Standard — kiegyensúlyozott komplexitás a legtöbb témához.
