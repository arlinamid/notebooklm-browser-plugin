# Infografika — Összehasonlítás

**Forrás:** Custom - Validated
**Tájolás:** Landscape (16:9)
**Részletesség:** Standard

## Irányítóprompt

```
Osztott képernyős összehasonlító elrendezés, nagy kontrasztú duotón paletta.
Bal oldal: hideg kék tónusok. Jobb oldal: meleg narancs tónusok.
Középső függőleges tengely az összehasonlítási szempontokat sorolja fel.
Letisztult rácsigazítás, talp nélküli betűtípus, minimális díszítés.
Ikonok az egyes jellemzők megjelenítéséhez, pipák a képességeknél.
```

## Swiss változat

```
Egymás melletti összehasonlító infografika, svájci dizájn-esztétika.
Két elkülönített színzóna, középső elválasztóval.
Strukturált sorok az egyes összehasonlítási szempontoknál.
Ikonok és vizuális jelzők a szöveges leírások helyett.
Professzionális monokróm kiemelőszínekkel a megkülönböztetéshez.
```

## Felhasználási eset (adatmérnökség)

Két megközelítés vagy technológia összehasonlítása. Ideális a következőkhöz:
- ETL vs ELT (igazított szempontokkal)
- Batch vs Streaming feldolgozás
- Eszközök összehasonlítása (dbt vs Dataform, Airbyte vs Fivetran)

**Validálva tesztelve:** AWS - ETL vs ELT

**Megjegyzés:** A "split-screen" direktíva nagyon olvasható 4 oszlopos struktúrát eredményez tökéletes színdifferenciálással.
