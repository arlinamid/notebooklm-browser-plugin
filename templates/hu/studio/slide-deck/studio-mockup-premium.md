# Studio Mockup Prémium

**Forrás:** Original Library
**Formátum:** Slide Deck
**Nehézség:** Advanced

## Prompt

```
# presentation_design_spec_premium_mockup_jp.yaml
# Source: Uploaded Images (Apple Device Mockups with Modern UI)
# Style: Premium Mockup / Modern UI / Clean Tech
# Concept: "Showcase in Perfection"

Global Design Settings:
  Tone: "Kiváló minőségű, fejlett, letisztult, kifinomult, professzionális"

Color Palette:
  Background:
    - "#FFFFFF (tiszta fehér)"
    - "#F5F5F7 (nagyon világosszürke, stúdiószerű)"
    - "#000000 (koromfekete, dianként váltakozva)"
  Accent Colors:
    - "#8D59E9 (Electric Purple – fő akcióSzín)"
    - "#EBE021 (Acid Yellow – kiemelési pontok és jelvények)"
  Sub Colors:
    - "#D8E2EC (halvány kékesszürke kártyákhoz és alapterületekhez)"
    - "#2D2D2D (szén szöveghez és UI-elemekhez)"

Visual Identity:
  Devices: "Apple-termékek kiváló minőségű 3D makettjei (Studio Display, MacBook Pro, iPad, iPhone 16 Pro)"
  UI Screen Design:
    Background: "Koromfekete (#000000) vagy élénk átmenetek (lila, sárga, narancs)"
    Typography: "Extra félkövér sans-serif (pl. Helvetica Now Display Bold) fehér színben"
    Layout: "Kártyaalapú UI, rácselrendezés, túlméretezett számok"

Typography:
  Headings: "Tömbös sans-serif betűtípus nagy méretaránnyal. Angol nagy, (a felhasználó által kért nyelvű) kis méretű."
  Body: "Vékony szürke gótikus betűtípus. Kiemeléshez szöveget helyezz el fekete (#000000), fehér (#FFFFFF), Electric Purple (#8D59E9) vagy Acid Yellow (#EBE021) háttéren."

Common Layout Rules:
  - "Stúdióvilágítás: Lágy árnyékokat alkalmazz az eszközök alatt a valósághű stúdióhatás érdekében."
  - "Képernyő mint főszereplő: Az eszközképernyők a dia területének 70–80%-át foglalják el."
  - "Vágás/Túlnyúlás: Az eszköz egy részét engedd túlnyúlni a dia határain a léptékezés és hatás érdekében."

Layout Variations:
  - Type: "Hero Display"
    Design: "Helyezd középre a Studio Display-t fekete hátterű UI-val és ultra-félkövér fehér főcímmel (pl. 'STUDIO DISPLAY MOCKUPS'). Nagy angol cím, kis (felhasználó által kért nyelvű) alcím."

  - Type: "Floating Mobile"
    Design: "Lebegtesd az iPhone makettjét a levegőben, alkalmazás-UI megjelenítésével. Adj hozzá elmosódott, kiemelőszínű megvilágítást mögé."

  - Type: "Grid Interface"
    Design: "MacBook képernyőjén belül rendezz el színes UI-kártyákat (lila, cián, sárga) letisztult rácsban."

  - Type: "Dark Mode Presentation"
    Design: "Eszközképernyő sötét módban. Fehér + Acid Yellow kombinációt alkalmazz nagy kontrasztú kiemeléshez."

  - Type: "Angle Shot"
    Design: "Mutasd az eszközt átlós szögből, a vékonyságát és textúráját kiemelve. Adjon hozzá spec-információkat az üres területen."

  - Type: "Split Screen"
    Design: "Bal: egy eszközmakett fele. Jobb: nagy tipográfia. Biztosítsd a zökkenőmentes kapcsolatot a képernyőszínek és a háttér között."

  - Type: "Card Grid (Text Only)"
    Design: "Az eszköz UI-elrendezését terjeszd ki az egész diára. Fehér vagy lila kártyákat helyezz el #D8E2EC háttéren."

  - Type: "Big Typography"
    Design: "Fekete háttér (#000000) masszív fehér ultra-félkövér szöveggel. Angol nagy, (a felhasználó által kért nyelvű) nagyon kicsi. Vékony fehér elválasztókkal."

  - Type: "Split UI"
    Design: "Bal: nagy számok ('16', '3680'). Jobb: leíró szöveg. Kártya-szerű keretbe csomagolva."

  - Type: "Feature List Card"
    Design: "Fehér háttér lekerekített szürke sávval (#F5F5F7). Belül: igazított ikonok + szöveg, nagy angol cím, kis (a felhasználó által kért nyelvű) alcím."

Design Rules:
  - "Nagy felbontás: Minden makett képnek kiváló felbontásúnak kell lennie, pixelesedés nélkül."
  - "Tükröződés és fényesség: Adj vissza valósághű képernyő-tükröződéseket a prémium hatáshoz."
  - "UI-térköz: Az eszköz UI-ja is kövessen tágas, minimális elrendezést."
```
