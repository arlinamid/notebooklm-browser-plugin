# Éles Élű Minimalizmus

**Forrás:** Original Library
**Formátum:** Slide Deck
**Nehézség:** Advanced

## Prompt

```
# presentation_design_spec_minimal_jp.yaml
# Style: Refined Minimal Portfolio
# Characteristics: Top-left navigation, aesthetics of whitespace, grid-based layout

Global Design Settings:
  Tone: "Professzionális, architektúrai, éles élű minimalizmus"
  Color Palette:
    Base: "#E9E9E9 (világosszürke) vagy #FFFFFF (fehér)"
    Text: "#000000 (koromfekete) vagy #333333 (sötétszürke)"
    Accent: "#000000 (fekete) – félkövér vonalakhoz és kiemelt szöveghez"
    Special: "Sötét mód (fekete háttér) – hangsúlyos diákhoz"
  Typography:
    Headings: "Angol sans-serif (pl. Helvetica Now, Inter). Félkövér és dekoratívan elhelyezve."
    Body: "(a felhasználó által kért nyelvű) gótikus betűtípus. Kis méret, bőséges betűközzel és sortávolsággal." A nyelv az legyen, amit a felhasználó megadott.
  Common Layout Rules:
    Navigation: "Minden dia bal felső (vagy jobb felső) sarkában helyezz el egy kis szakaszszámot és címet, pl. '01. INTRODUCTION'."
    Grid: "Szigorú rácsrendszert alkalmazz az elemek igazításához."
    Whitespace: "Szándékosan hagyj nagy területeket üresen (negatív tér) a luxusérzet megteremtéséhez."

Layout Variations (Catalog):
  - Type: "Title Typography"
    Design: "Szórt elrendezés. Helyezz el véletlenszerűen díjbadge-eket vagy kulcsszavakat bélyegszerűen. A középső cím kis méretű, félkövér és visszafogott."

  - Type: "Text + Data Emphasis"
    Design: "Aszimmetrikus osztás. Bal oldalon elbeszélő szöveg, jobb oldalon túlméretezett számok (fekete). Vékony elválasztóvonalakkal."

  - Type: "Card Grid"
    Design: "Sűrűn elhelyezett rács. Kiváló minőségű képek webes interakciós érzéssel, például egérátúszáskor megjelenő szöveg."

  - Type: "Full-Screen Graphic"
    Design: "Irodai enteriőr fotó teljes képernyőn (vagy több mint felén). Betonfelületek, Mac-ekkel teli asztalok. Csökkentett telítettség a hűvös tónus érdekében. Nagyon kis felirat bal alul."

  - Type: "Photo + List Split"
    Design: "50:50 osztás. Bal: építészeti vagy absztrakt fotó. Jobb: adatlista (félkövér fejlécek + könnyű súlyú leírások). Vállalati információkhoz használatos. Félkövér tételnevek, vékony törzsszöveg. Bőséges térköz; kerüld a zsúfoltságot." A nyelv az legyen, amit a felhasználó megadott.

  - Type: "Minimal Map"
    Design: "Sziluett stílusú térkép. Világosszürke háttér fehér térképpel. Ultra-vékony magyarázóvonalak a helyek jelölésére."

  - Type: "Vertical Timeline"
    Design: "Függőleges tengely. Egyetlen vékony vonal, amelyről szöveg ágazik el jobbra és balra. Tiszta kronológiai sorrend."

  - Type: "Bubble Chart / Venn Diagram"
    Design: "Dróthálós stílus. Fekete háttér vékony fehér vonalakkal. Félig átlátszó, átfedő körök."

  - Type: "Dialogue (Chat Style)"
    Design: "Minimális párbeszédes formátum. Nem képregény-buborékok, hanem egyszerű szövegblokkok félkövér nevekkel."

  - Type: "Chronological List"
    Design: "Ritmikus lista. Bal oldalon nagy évszámok (pl. 2024), jobb oldalon leírások. Erős betűméret-kontraszt."

  - Type: "Dark Mode Diagram"
    Design: "Intellektuális tech-esztétika. Fekete háttér vékony fehér vonalakkal, amelyek csomópontokat kötnek össze. Csillagkép- vagy hálózatszerű megjelenés. Finom vonalakkal összekötött körök vagy geometriai minták, a technológia és kreativitás találkozásának kifejezéseként."

  - Type: "3-Step Columns"
    Design: "Tipográfia-vezérelt oszlopok. Nagy számok (01, 02, 03) pillérként szolgálnak. Ikonok nélkül, tipográfiai kontrasztra támaszkodva."

  - Type: "Logo Grid"
    Design: "Fekete-fehér rács. Minden logót feketére vagy szürkére alakítva, szigorú rácsban igazítva."

  - Type: "Two Columns (Problem vs Solution)"
    Design: "Éles kontraszt. Vastag fekete függőleges vonal választja el a 'Problémát' és a 'Megoldást'. Szöveg blokkos formában igazítva."

  - Type: "Centered Layout (Dark Mode)"
    Design: "Filmszerű. Kis videó-előnézet vagy kulcsvizuál középre helyezve fekete háttéren. Érzelmes angol tagline hozzáadása."

  - Type: "Formula / Flow Diagram"
    Design: "Matematikai stílus. 'A × B = C' formájú kifejezések nagy talpas betűtípussal. Minimális nyilak."

  - Type: "Arrow Steps"
    Design: "Lineáris folyamat. Szöveg nagy nyilakba helyezve. Magas kontraszt (fekete nyilak fehér szöveggel)."

  - Type: "Chart"
    Design: "Precíz adatvizualizáció. Vékony vonalú grafikonok kis fekete pontokkal a végeken. Tudományos műszer-szerű megjelenés."
```
