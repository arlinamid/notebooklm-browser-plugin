# Manga Stílus

**Forrás:** Original Library
**Formátum:** Slide Deck
**Nehézség:** Beginner

## Prompt

```
# presentation_design_spec_manga_comic.yaml
# Style: Manga / Comic Book
# Concept: "Story as Structure — Information through Sequential Art"

Global Design Settings:
  Tone: "Dinamikus, kifejező, narratív, energikus, mégis olvasható"

  Color Palette:
    Base: "#FFFFFF (tiszta fehér — képregénypapír)"
    Text: "#000000 (erős tintafekete)"
    Accent:
      - "#FF4444 (hatáspiros — sebességvonalak, kiemelés, veszély)"
      - "#FFD700 (kiemelő sárga — csillagok, fókuszpillanatok)"
    Tones: "Raszterpontmintázatok és képernyőtónusok az árnyékolt területekhez"
    ColorRule: "Szelektív pontszín: diánként maximum 1–2 kiemelőszín. Alap: fekete, fehér, egy szín."

  Typography:
    Headings: "Félkövér, kézírásos vagy manga-stílusú betűtípus. Csupa nagy vagy vegyes nagybetűs, hatásos."
    Body: "Képregény-stílusú vagy lekerekített sans serif a szövegbuborékokban."
    SFX: "Nagy hanghatás-tipográfia (POW!, ZOOM!, HA!) grafikus elemként használva."
    Numbers: "Félkövér, túlméretezett — vizuális elemként kezelve, nem egyszerű feliratként."
    Language: "A szöveg nyelve legyen az, amit a felhasználó megadott a promptban."

  Common Layout Rules:
    Panels: "Osszd fel a diákat 2–4 manga-panelre, vastag fekete gutterekkel elválasztva (4–6pt keretek)."
    SpeechBubbles: "Szövegbuborékokat és gondolatfelhőket használj a kulcsinformációkhoz, idézetekhez és adatpontokhoz."
    ReadingFlow: "Balról jobbra, fentről lefelé haladó panel-sorrend (nyugati manga-konvenció)."
    SpeedLines: "Radiális sebességvonalakat adj a kulcselemek mögé vizuális hatás és fókusz érdekében."
    ScreenTones: "Raszterpontmintázatokat használj árnyékolt területekhez — nem tömör szürke kitöltéseket."

Layout Variations:
  - Type: "Cím / Borítóoldal"
    Design: "Teli vérzésű kompozíció. Egy központi karakter vagy fogalom-illusztráció uralja a diát. Nagy manga-stílusú cím-tipográfia, drámai sebességvonalakkal kifelé sugározva. Kis alcím egy tiszta téglalap alakú feliratdobozban alul."

  - Type: "Négypaneles Történetrács"
    Design: "Klasszikus 2×2 panelrács. Minden panel egy lépést, fogalmat vagy adatpontot mutat. A panelek mérete a fontosságot tükrözi — a legkritikusabb panel a legnagyobb. Vastag fekete keretek (4–6pt) a panelek között."

  - Type: "Szövegbuborék-adatok"
    Design: "Több, különböző méretű, egymást átfedő szövegbuborék, mindegyikben egy-egy kulcstény, idézet vagy statisztika. Karakterek vagy illusztrált személyiségek mondják az adatokat. Háttér egy egyszínű lapos szín vagy képernyőtónus."

  - Type: "Nagy Hatású Oldal"
    Design: "Egyetlen, teli méretű panel drámai jelenettel. Egy hatalmas szám vagy mondat középre helyezve. Sebességvonalak sugároznak ki belőle. Egy kis feliratdoboz a sarokban adja meg a kontextust."

  - Type: "Chibi Magyarázó"
    Design: "Egy kis deformált (chibi) karakter vezeti végig a nézőt egy lépésről-lépésre folyamaton. Minden lépés egy téglalap feliratdobozban van jelölve. A karakter kifejezően reagál (meglepett, elszánt, ünneplő) minden pontnál."

  - Type: "VS Csata-elrendezés"
    Design: "Képernyő átlósan osztva. Bal: A lehetőség karakterével vagy ikonjával. Jobb: B lehetőség. Villám vagy fogazott vonal választja el őket. Az előnyök és hátrányok feliratdobozokban vannak mindkét oldalon."

  - Type: "Idővonal / Fejezet-folyamat"
    Design: "Vízszintes panelcsík a dia teljes szélességén. Minden panel = egy időszak vagy fázis. Manga sebességvonal-stílusú iránymutató nyilakkal összekötve. A kulcsmozzanatok karakterreakció-panelként láthatók."

Design Rules:
  - "Körvonalak: Minden illusztrációs elem vastag, magabiztos fekete tintavonalat használ (minimum 3–5pt). Nem vékony hajszálvonalak."
  - "Képernyőtónusok: Raszterpontmintázatokat használj árnyékolt területekhez — soha nem tömör szürke kitöltést."
  - "Panelkeretek: A panelek közötti gutterek mindig tömör feketék (4–6pt). A panelek határozzák meg a struktúrát."
  - "Kifejezések: A karaktereknek egyértelmű, eltúlzott érzelmi reakciókkal kell rendelkezniük — semleges arcok nincsenek."
  - "Színmértékletesség: Diánként maximum 2 pontszín kiemelő. Az alap fekete, fehér és egy szín."
  - "Nem vállalati sablonok: Nem gradiens háttér, nem árnyékvetés, nem PowerPoint alapstílusok."
  - "Generálás: Készíts összefüggő, több diás bemutatót, ahol minden dia manga-oldalként funkcionál. Tartsd fenn a következetes karakterdizájnt, panelkeret-stílust és tipográfiai hangvételt végig."
```
