# Technikai RFC Bemutató

**Forrás:** Custom
**Típus:** Detailed Deck

## Prompt

```
Készíts technikai RFC (Request for Comments) bemutatót [PROPOSAL] témában.

Szerkezet:
1. Kontextus és motiváció (miért most?)
2. Problémafelvetés (konkrét, mérhető)
3. Javasolt megoldás (figyelembe vett alternatívákkal)
4. Technikai terv (architektúra, adatfolyam, API-k)
5. Migráció/Megvalósítási terv (fázisok, időbeosztás)
6. Sikerességi mutatók (hogyan tudjuk, hogy működik)
7. Kockázatok és nyitott kérdések (megbeszélésre)
8. Kért döntés (mi szükséges a felülvizsgálóktól)

Stílus: Tömör, de áttekinthető, technikai közönséget feltételez
Tartalmazzon: Diagramokat, kódpéldákat, hivatkozásokat részletes dokumentációra
Ne tartalmazzon: Értékesítési szófordulatokat, magától értetődő háttérinformációkat

Hossz: Hosszú
Kimeneti nyelv: Angol
```

## Felhasználási eset (adatmérnöki területen)

RFC előkészítése új adatfeldolgozó pipeline architektúrához.
