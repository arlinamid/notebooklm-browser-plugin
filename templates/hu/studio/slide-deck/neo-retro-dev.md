# Neo-Retro Fejlesztői Bemutató

**Forrás:** Original Library
**Formátum:** Slide Deck
**Nehézség:** Advanced

## Prompt

```
Általános stílusnév
Neo-Retro Dev Deck / Pixel-Infographic Editorial
1. Alapvető vizuális identitás
Esztétika:
Retro-futurista, fejlesztőközpontú, szerkesztői infografika stílus
Hangulatában hasonlít:
90-es évek számítógépes kézikönyvei
modern AI fejlesztői eszközök marketingje
pixel-art találkozik a startup slide deck-kel
Hangulat:
Magabiztos, játékos, véleményes, enyhén lázadó
„Építők magyarázzák el a jövőt, nem adnak el szólamokat"
2. Vászon és háttér
Világos krém / törtfehér négyzetrácsolt papír háttér
Finom négyzetrács (mérnöki noteszfüzet érzés)
A rácsvonalak nagyon halványak, soha nem zavaróak
A diák annotált mérnöki vázlatoknak hassanak, ne vállalati PowerPoint-nak
3. Tipográfiai rendszer
Elsődleges főcím-betűtípus:
Félkövér, nehéz sans-serif erős geometriával
Fekete szöveg
Enyhén keskenyített
Nagy kontraszt a háttérrel szemben
Nyelvkeverés:
A főcímek lehetnek „a felhasználó által megadott nyelv" + angol keveréke
Angol alcímkék a „felhasználó által megadott nyelvű" főcímek alatt (kis méret, letisztult)
Hierarchia:
Hatalmas félkövér főcím-blokkok
Közepes méretű alcímek
Kis magyarázó feliratok ikonok vagy dobozok alatt
4. Színpaletta (szigorú)
Nagy kontrasztú blokkokat alkalmazz vastag fekete körvonalakkal:
Élénkrózsaszín (ágens / agy / intelligencia fogalmak)
Élénksárga (szerkesztő / kód / eszközök)
Cián / világoskék (böngésző / web / végrehajtás)
Fekete (szöveg, keretek)
Fehér / krém (háttér)
Minden szakasz = egy domináns színblokk
5. Elrendezési nyelv
Egymásra épülő moduláris blokkok
Vastag fekete keretű téglalapok
Enyhe átfedések megengedettek (szándékos, kollázs-szerű)
Vízszintes sávok szakaszfejlécekhez
Kártyaalapú struktúra lépésekhez, fejlődési vonalakhoz, rétegekhez
A diák összerakottnak hassanak, nem tökéletesen igazítottnak
(kontrollált tökéletlenség)
6. Ikonográfia és grafika
Pixel-art stílusú ikonok, például:
Rakéta
Robot / ágens
Fogaskerék
Kód zárójelpár
Böngészőablak
Csevegőbuborékok
Az ikonok legyenek:
Alacsony felbontású
8-bit vagy 16-bit ihletettsége
Lapos színek, fekete körvonal
Díszítő elemek:
Kis fogaskerekek
Nyilak
Csúcsos zárójelek <>
Pixel szikrák / mozgásvonalak
7. Tartalmi minták (nagyon fontos)
A. „Rendszerarchitektúra" diák
Egymásra épülő rétegek feliratokkal
Minden réteg:
Egy szín
Egy ikon
Egy félkövér cím
Egy rövid magyarázó sor
Példa struktúra:
Agent Manager
AI Editor
Agent-Controlled Browser
B. „Fejlődés / Idővonal" diák
Balról jobbra haladó folyamat
Minden lépés saját dobozban:
Automatikus kiegészítés
Chat
Ágens
Új modell (kiemelve)
Az utolsó lépés legyen vizuálisan nagyobb és színesebb
C. „Kiáltvány / Tézis" diák
Egy hatalmas főcím keretezett blokkban
Minimális szöveg
Játékos ikonokkal körülvéve
Kijelentésnek hasson, nem dokumentációnak
8. Szöveg hangneme
Rövid
Kijelentő
Enyhén véleményes
Semmi marketingszlogen
Jó példa:
„Az ágensek önállóan hajtják végre a feladatokat"
„A kódolás belép az ágens-korszakba"
Rossz példa:
„Felhasználókat felhatalmazó, élvonalbeli megoldások"
9. Kerülendők
Semmilyen átmenetes szín (gradient)
Sem realisztikus fényképek
Sem lágy árnyékok
Sem vállalati sablonok
Sem minimalizmus önmagáért valóan
Ez kifejező stílus, nem visszafogott
10. Generálási utasítás (kritikus)
Generálj több diát pontosan ezt a vizuális rendszert követve.
Tartsd fenn a következetességet:
rácsos háttér
színhasználat
ikonok stílusa
tipográfiai hierarchia
A diák egyetlen összefüggő bemutatóhoz tartozzanak, ne különálló plakátokhoz.
```
