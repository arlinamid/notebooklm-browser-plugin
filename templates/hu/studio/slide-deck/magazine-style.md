# Magazin Stílus

**Forrás:** tetumemo
**Formátum:** Slide Deck
**Nehézség:** Advanced

## Prompt

```
# Utasítások
Kérlek, rendezd össze a [Forrásadatokat] egy [érett-aranyos, kifinomult magazinstílusú szerkesztőségi dizájnban].

# Elrendezési követelmények
・Helyezz el egy nagy kivágott fotót az alanyról (modellnő vagy termék) középen, a mozgást vagy a pózát kiemelve
・Helyezz el szövegbuborékokat vagy szövegdobozokat aszimmetrikusan a bal és jobb margókon
・A fő pontokat oszd el kis szakaszokban, amelyeket „NO.1", „NO.2" stílusban számozz
・Helyezz el egy függőleges szövegdobozt (fehér háttérrel) a képernyő szélén (pl. jobb oldalon), és írj bele egy figyelemfelkeltő szlogent
・Helyezz el L-alakú vonalakat a négy sarokban, amelyek vágásjelekre emlékeztetnek, hogy plakát- vagy oldalszerű hatást kelts
・Formátum: fekvő tájolás (4:3 vagy 3:2), nagy felbontás
・A szöveg nyelve legyen az, amit a felhasználó a promptban megadott.

# Tervezési követelmények
・Háttérszín: matt, visszafogott, elegáns rózsaszín (dusty pink vagy shell pink)
・Főkép: Teljesen kivágott (mélyen retusált) háttér nélküli fotó
・Dekoratív elemek: kézzel rajzolt stílusú szövegbuborékok, egyszerű körök és vékony egyenes vonalak kiemelésként
・Betűtípusok: a főcímek kifinomult Gothic vagy Mincho betűtípusban, a megjegyzések kézzel rajzolt stílusú betűtípussal a könnyed hatás érdekében
・A szövegbuborékokban vagy szövegdobozokban a forrásadatok kulcspontjait egy fiatal nő érzelmes, kézzel írt szövegével fejezd ki
・Színpaletta: matt rózsaszín a háttérhez, sötétszürke vagy fekete a szöveghez, fehér a kiemelőelemekhez
・Általános hangulat: érett-aranyos, nőies, trendi, csiszolt tónus

# Kerülendő kifejezések
・Túlzottan gyerekes, élénk színek, túlságosan csillogó átmenetek, erős árnyékok
```
