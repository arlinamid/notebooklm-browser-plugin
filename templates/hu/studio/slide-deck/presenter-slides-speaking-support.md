# Előadói Diák - Beszédtámogatás

**Forrás:** vibeproductmarketing
**Típus:** Presenter Slides

## Prompt

```
Készíts előadói diákat [AUDIENCE: csapat standup / tudásmegosztás] számára.

Szerkezet:
- Címdia kulcsüzenettel
- 3-5 tartalmi dia (dianként egy gondolat)
- Vizuális hangsúly diagramokon/grafikonokon
- Előadói jegyzetek beszédpontokkal
- Záródia következő lépésekkel/teendőkkel

Stílus: Letisztult, vizuális, kevés szöveg (felsoroláspontok csak az előadó számára)
Tartalmazzon: Megbeszélésre ösztönző felvetéseket, a közönségnek feltehető kérdéseket
Ne tartalmazzon: Teljes mondatokat a diakon, sűrű információhalmazokat

Hossz: Rövid
Kimeneti nyelv: [Magyar/Angol]
```

## Felhasználási eset (adatmérnöki területen)

Gyors bemutató csapaton belüli tudásmegosztáshoz.
