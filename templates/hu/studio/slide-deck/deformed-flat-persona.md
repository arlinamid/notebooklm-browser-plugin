# Deformált Lapos Persona

**Forrás:** Kawai
**Formátum:** Slide Deck
**Nehézség:** Beginner

## Prompt

```
# Stílus:
- Lapos színek

# Illusztráció:
- Enyhén deformált alakú személy
- Vastag körvonalak

# Szín:
- Lágy tónusok, egy kis fehérrel keverve
- Legfeljebb 3 szín engedélyezett

# Körvonal:
- Vastag

# Háttér:
- Egységes lapos szín, 1 színnel
```
