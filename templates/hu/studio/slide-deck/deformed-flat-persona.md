# Deformált Lapos Persona

**Forrás:** Kawai
**Formátum:** Slide Deck
**Nehézség:** Beginner

## Prompt

```
# presentation_design_spec_deformed_flat_persona.yaml
# Style: Deformed Flat Persona
# Concept: "Character-Led Communication — Human Warmth through Illustration"

Global Design Settings:
  Tone: "Barátságos, könnyen megközelíthető, meleg, egyértelmű, karaktervezérelt történetmesélés"

  Color Palette:
    Base: "Egyetlen tömör lapos pasztell szín diánként — rotálva a teljes bemutatón"
    PastelPalette:
      - "#FFE0B2 (meleg barack)"
      - "#E8F5E9 (mentazöld)"
      - "#E3F2FD (égkék)"
      - "#F3E5F5 (lágy levendula)"
      - "#FFF9C4 (halvány sárga)"
    Text: "#2D2D2D (meleg szénszürke — nem nyers fekete)"
    Outline: "#1A1A1A (vastag, barátságos tintafekete — minden karakter- és illusztrációs körvonal)"
    Rule: "Diánként maximum 3 szín: háttér + karakterszín(ek) + egy kiemelő. Kivétel nincs."

  Typography:
    Headings: "Lekerekített, barátságos félkövér sans serif — könnyen megközelíthető, nem vállalati."
    Body: "Letisztult lekerekített sans serif. Kis méret, jó sortávolság."
    Numbers: "Félkövér, nagy, játékos — vizuális karakterekként kezelve."
    Language: "A szöveg nyelve legyen az, amit a felhasználó megadott a promptban."

  Common Layout Rules:
    Characters: "Minden emberi alak deformált arányokat használ: nagy fej (a testmagasság 50–60%-a), kis lekerekített test, egyszerűsített végtagok."
    Outlines: "Minden karakter és illusztrációs elem vastag fekete körvonalat használ (4–6pt). Soha nem vékony vagy hajszálvonal."
    ColorCount: "Szigorú maximum 3 lapos szín diánként. Sem gradient, sem textúra, sem árnyékvetés."
    Expressions: "A karakterek mindig egyértelmű, eltúlzott arckifejezéseket mutatnak — boldog, meglepett, gondolkodó, magyarázó. Semleges arc nincs."
    BackgroundRotation: "A pasztell háttérszínt diáról-diára váltsd ritmus és vizuális folyamat érdekében."

Layout Variations:
  - Type: "Bemutatkozás / Cím"
    Design: "Tömör pasztell háttér. Barátságos deformált karakter jobbra középen, integetve vagy a cím felé mutatva. Cím nagy lekerekített félkövér betűkkel balra. A karakter szájából kis szövegbuborék rövid üdvözlő mondattal."

  - Type: "Persona Profil"
    Design: "Letisztult kártyaelrendezés pasztell háttéren. Nagy karakterillusztráció bal oldalon (a diaszélesség ~40%-át elfoglalva). Név félkövér főcímben, szerepkör vagy leírás szövegtörzsben jobbra. Kis ikon-cimkék az attribútumokhoz alul."

  - Type: "Magyarázó Karakterrel"
    Design: "A karakter egy folyamat-diagram mellett vagy belsejében áll. Szövegbuborékok vagy gondolatfelhők tartalmazzák a kulcsmagyarázatokat. A karakter póza és kifejezése tükrözi a tartalmat (gondolkodó = kéz az állán, magyarázó = felfelé mutat)."

  - Type: "Lépésről-lépésre Útmutató"
    Design: "3–4 lépés vízszintesen megjelenítve. Minden lépés: számozott kör, kis karakter a megfelelő pózban, félkövér lépéscím, 1–2 sor szöveg. A karakterkifejezések lépésről-lépésre változnak a haladás megmutatásához."

  - Type: "Előtte / Utána"
    Design: "Dia függőlegesen osztva. Bal (előtte): karakter frusztrált vagy negatív pózban, tompított vagy sötétebb tónusú színnel. Jobb (utána): karakter boldog vagy ünneplő pózban, fényesebb pasztellel. Egyszerű nyíl átmenet a panelek között."

  - Type: "Csapat / Szereplőgárda"
    Design: "Több karakter vízszintes sorban elrendezve, mindegyik enyhén eltérő színnel vagy jelmezmotívummal. Név és szerepkör minden karakter alatt. Egyenletes elosztás. Szereplőgárda-bemutató vagy csapatoldal érzete."

  - Type: "Adat Érzéssel"
    Design: "Egy nagy szám vagy statisztika prominensen kijelezve. Egy karakter reagál rá — ugrik ha pozitív, összeesik ha negatív, töpreng ha összetett. Rövid magyarázó szöveg a karakter mellett."

Design Rules:
  - "Fej-Test Arány: A karakterfejek mindig a teljes testmagasság 50–60%-át teszik ki. Ez az alapvető deformált esztétika."
  - "Körvonal-vastagság: Minden illusztrációs körvonal 4–6pt vastag. Soha nem vékony vagy hajszálvonal."
  - "Háromszínes maximum: Háttér + karakterszín(ek) + egy kiemelő = maximum 3 szín. Kivétel nincs."
  - "Nem gradient: Minden kitöltés 100% tömör lapos szín. Nem árnyékolás, nem gradient, nem árnyékvetés."
  - "Kifejezések számítanak: A semleges arcú karakter helytelen. Minden karakternek szándékos, egyértelmű érzelmet kell mutatnia."
  - "Generálás: Készíts összefüggő, több diás bemutatót, ahol deformált lapos karakterek vezetik végig a nézőt a tartalmon. Tartsd fenn a következetes karakterarányt, körvonal-vastagságot és lapos szín-fegyelmet végig."
```
