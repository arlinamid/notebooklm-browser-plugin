# Részletes Bemutató - Önálló Dokumentum

**Forrás:** vibeproductmarketing
**Típus:** Detailed Deck

## Prompt

```
Készíts részletes bemutatót [AUDIENCE: technikai felülvizsgálati bizottság] számára.

Szerkezet:
1. Problémafelvetés (1 dia)
2. Jelenlegi állapot elemzése (2-3 dia)
3. Javasolt megoldás architektúrával (3-4 dia)
4. Megvalósítási ütemterv (2 dia)
5. Kulcsmutatók és sikerességi kritériumok (1 dia)
6. Kockázatok és kockázatcsökkentési intézkedések (1 dia)
7. Kérdések és válaszok / Megbeszélési pontok (1 dia)

Stílus: Professzionális, adatvezérelt, kevés szöveg dianként
Tartalmazzon: Kódrészleteket ahol releváns, architektúra-diagramok leírásait
Ne tartalmazzon: Sablonos stock képeket, motivációs idézeteket

Hossz: Alapértelmezett
Kimeneti nyelv: [Magyar/Angol]
```

## Felhasználási eset (adatmérnöki területen)

Technikai bemutatók architektúra-felülvizsgálathoz vagy RFC-hez.
