# Tech Art Neon

**Forrás:** Original Library
**Formátum:** Slide Deck
**Nehézség:** Advanced

## Prompt

```
# presentation_design_spec_neon_collage_jp.yaml
# Source: Uploaded Image (1bce448a...jpg)
# Style: Constructivism / Tech-Art / Avant-Garde
# Concept: "Architecture of Intelligence"

Global Design Settings:
  Tone: "Avantgárd, strukturális, intellektuális, művészi, jövőorientált"

  Visual Identity:
    Background Color: "Meleg szürke / bézs (#E0E0D0) — matt, papírszerű textúra"
    Text Color: "Sötétszürke (#333333) — nem tiszta fekete"
    Accent Color: "Neon sárga (#DFFF00) — geometriai alakzatokhoz"
    Line Color: "Ultra-vékony szürke vonalak (0.5pt), hasonlóan az építészeti tervrajz-vonalakhoz"

  Typography:
    Headings: "Talpas (Didot, Bodoni) és talp nélküli (Helvetica) betűtípusok keveréke"
    Body: "Kis méretű szöveg, szigorúan a rácshoz igazítva"
    Numbers: "Írógép-stílusú betűtípusok (Courier New)"

Slide Composition Patterns:

  # --- Cover / Concept ---
  Section: "Borító / Koncepció"
  Layout Type: "Triple Collage"
  Design Details:
    - "Struktúra: A képernyő három részre osztva rácsvonalakkal"
    - "Vizuális elemek: Fekete-fehér kivágott portrék"
    - "Kiemelők: Neon sárga körök és négyzetek az alany mögött vagy előtt"
    - "Dekoráció: Koncentrikus körök (célpont-szerű) a fókuszpont köré helyezve"

  # --- Analysis / Diagram ---
  Section: "Elemzés / Felbontás"
  Layout Type: "Technical Drawing"
  Design Details:
    - "Háttér: Finom rács, hasonlóan a kockás papírhoz"
    - "Fő elem: Termékek vagy fogalmak vonalrajza/dróthálós illusztrációja, részben neon sárgával kitöltve"
    - "Annotációk: Vékony vezérvonalak tervrajz-stílusú feliratokkal (pl. Fig.1, Fig.2)"

  # --- Process / Flow ---
  Section: "Folyamat / Áramlás"
  Layout Type: "Geometric Connection"
  Design Details:
    - "Elemek: Neon sárga körök, négyzetek, háromszögek"
    - "Kapcsolatok: Pontozott vagy szaggatott vonalak az alakzatok között a logikai folyamat kifejezéséhez"
    - "Fényképek: Az alakzatokat maszkként használva fekete-fehér fotókat ágyazz beléjük"

  # --- Data / Infographic ---
  Section: "Adat / Infografika"
  Layout Type: "Radar Chart Art"
  Design Details:
    - "Diagram: Nagy pókháló-szerű radar diagram az egész dián"
    - "Adatvonalak: Vastag neon sárga ábrázolási vonalak"
    - "Háttér: Finom textúra, régi térképekre vagy tervrajzokra emlékeztető"

Design Rules:
  - "Színkorlátozás: Csupán három fő színt alkalmazz — bézs (háttér), fekete-fehér (fotók), neon sárga (geometria)"
  - "Kollázs: Soha ne helyezd a fotókat egyszerű téglalapként; mindig vágd ki az alanyt tisztán"
  - "Rétegelési sorrend: Rácsos háttér → neon geometriai alakzatok → fekete-fehér kivágott fotó → előtéri szöveg"
```
