## Prompt

```text
Készíts összehasonlító elemzési táblázatot a forrásanyagból, amely összehasonlítja a legfontosabb tárgyalt fogalmakat, entitásokat vagy megközelítéseket.

Tábla szerkezete:
- Minden oszlop egy összehasonlítandó fogalmat/entitást/megközelítést képvisel
- Minden sor egy értékelési dimenziót vagy szempontot képvisel

Javasolt dimenziók (tartalom szerint igazítva):
- Definíció / Alapcél
- Fő erősségek
- Fő gyengeségek / korlátok
- Mikor érdemes használni / ideális kontextus
- Bizonyíték minősége a forrásokban
- Összesített értékelés (1-5)

Utasítások:
- Minden cellában tömör, párhuzamos nyelvezetet használj
- Ahol alkalmazható, vastagon szedd a legjobb opciót minden sorban
- Az alján adj egy "Összefoglalás" sort általános ajánlással
- Vitatott vagy bizonytalan összehasonlításoknál lábjegyzeteket adj
```
