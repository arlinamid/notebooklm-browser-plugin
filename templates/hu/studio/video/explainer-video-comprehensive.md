# Magyarázó videó - Átfogó

**Forrás:** Wonder Tools
**Hossz:** 6-10 perc

## Prompt

```
Készíts egy MAGYARÁZÓ videót a következő célközönségnek: [TARGET AUDIENCE: data engineers evaluating [TOOL]].

Struktúra:
1. A probléma meghatározása (30 mp)
2. A megoldás áttekintése (1 perc)
3. Működési elv – technikai mélymerülés (3-4 perc)
4. Összehasonlítás alternatívákkal (2 perc)
5. Mikor érdemes / mikor nem érdemes használni (1 perc)
6. Az első lépések (1 perc)

Stílus: Technikai, de érthető, kódpéldákkal ahol releváns.
Vizuálisok: Architektúradiagramok, kódrészletek, összehasonlítótáblák.
Kihagyandó: Marketing szöveg, általános AI hype.
```

## Felhasználási eset (adatmérnökség)

Teljes körű magyarázó videó egy eszköz értékeléséhez POC előtt.
