# Technikai dokumentáció generátor

**Forrás:** Custom
**Nehézség:** Középhaladó

## Prompt

```
Technikai dokumentációs szakértőként dolgozol.

Készíts átfogó dokumentációt a következőhöz: [az adatpipeline-od/eszközöd/API-od]

Jelenlegi kód/konfiguráció: [töltsd fel vagy illeszd be]
Forrásanyagok: [feltöltött referencia-útmutatók, API dokumentumok]

Generáld le a következőket:
1. README.md (gyors kezdés, telepítés, használat)
2. Architekturális diagram leírása (komponensek és adatfolyamok)
3. API referencia (ha alkalmazható, a forrásokból vett példákkal)
4. Hibaelhárítási útmutató (a forrás dokumentumokból összegyűjtött gyakori problémák)
5. Bevált gyakorlatok (az összes forrásból kinyerve)

Kövesd a feltöltött forráspéldákban szereplő dokumentációs mintákat.
```

## Felhasználási eset (adatmérnökség)

Generálj professzionális dokumentációt kód és referenciamaterialokból.
