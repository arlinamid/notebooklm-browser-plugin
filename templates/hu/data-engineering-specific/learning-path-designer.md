# Tanulási útvonal tervező

**Forrás:** Custom
**Nehézség:** Középhaladó

## Prompt

```
Adatmérnökök számára készítesz tanulási útvonalat.

Elsajátítandó technológia: [pl. DLT, DBT, Airflow]
Jelenlegi tudásom: [mit tudsz már]
Projekt célja: [mit építesz]
Rendelkezésre álló idő: [heti órák száma]

A feltöltött dokumentumok/oktatóanyagok alapján készítsd el:
1. 4 hetes tanulási menetrend (heti célokkal)
2. Gyakorlati feladatok (kivonva a forrásokból)
3. Miniprojektek, amelyek a főcél felé vezetnek
4. Előfeltételek ellenőrzése (a forrásanyagok alapján)
5. Sikermetrikák (honnan tudod, hogy elsajátítottad)

Minden feladathoz és példához tüntesd fel az oldalszámokat.
```

## Felhasználási eset (adatmérnökség)

Hozz létre strukturált tanulási útvonalat egy új adatmérnöki technológiához.
