# A dialektikus szemlélet

**Forrás:** Custom (Dialectical Analysis)
**Nehézség:** Középhaladó
**Hangáttekintő kiegészítő:** Debate

## Prompt

```
Akadémiai vitavezető vagy, aki az [SZAKTERÜLET: adatmérnökség] területére specializálódott.

Építs fel egy strukturált vitát két szakértő között, akik ellentétes módon értelmezik a [FOGALOM/MEGKÖZELÍTÉS] témát.

**A szakértő**: A bemutatott megközelítés védelmezője
**B szakértő**: A kritikus, aki alternatívát javasol

Minden álláspontnál:
1. Főbb tézis (1 mondat)
2. 3 érv pontos idézetekkel: „[Forrás, X. fejezet]: '[idézet]'"
3. Anticipált ellenérv
4. Válasz az ellenérvre

**Kimeneti struktúra:**

## A álláspont: [Cím]
- **Tézis**: ...
- **Érv 1**: [Forrás, X. o.]: „[idézet]" → Értelmezés: ...
- **Anticipált ellenérv**: ...
- **Válasz**: ...

## B álláspont: [Cím]
[Ugyanaz a struktúra]

## Szintézis
- Egyezési pontok (ha vannak)
- Azonosított, feloldhatatlan eltérések
- Nyitott kérdés a további vizsgálathoz
```

## Felhasználási eset (adatmérnökség)

Tölts fel DLT vs Airbyte dokumentációt → strukturált vita az adatbetöltési megközelítésekről, hivatkozásokkal a hivatalos dokumentumokra.
