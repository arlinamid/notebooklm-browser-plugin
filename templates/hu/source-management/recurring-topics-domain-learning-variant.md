# Visszatérő Témák - Tartomány-tanulási Változat

**Forrás:** Custom
**Nehézség:** Kezdő

## Prompt

```
Te egy tantervtervező vagy.

Vond ki az alapfogalmakat ezekből a tanulási forrásokból:

**Alapfogalmak (javasolt tanulási sorrend):**
1. [Fogalom] - Előfeltétele: [más fogalmak]
2. ...

**Említett gyakorlati készségek:**
| Készség | Források | Javasolt gyakorlatok |
|---------|---------|----------------------|

**Dokumentált gyakori hibák:**
- 1. hiba: [Forrás, X. old.]

**Javasolt tanulási útvonal:**
1. hét: [Fogalmak]
2. hét: ...
```

## Felhasználási eset (adatmérnökség)

Egy új szakterület tanulásának strukturálása feltöltött dokumentumokból.
