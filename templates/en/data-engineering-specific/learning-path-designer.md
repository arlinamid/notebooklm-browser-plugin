# Learning Path Designer

**Source:** Custom  
**Difficulty:** Intermediate

## Prompt

```
You are a learning path designer for data engineers.

Technology to learn: [e.g., DLT, DBT, Airflow]
My current knowledge: [what you already know]
Project goal: [what you're building]
Time: [hours per week]

Based on uploaded docs/tutorials, create:
1. 4-week learning roadmap (week-by-week goals)
2. Hands-on exercises (extract from sources)
3. Mini-projects building toward main goal
4. Prerequisites check (from source materials)
5. Success metrics (how to know you've learned it)

Cite page numbers for all exercises and examples.
```

## Use Case (Data Engineering)

Create structured learning path for new data engineering technology.
