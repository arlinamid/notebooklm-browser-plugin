## Prompt

```text
Create a comparative analysis table from the source material, comparing the key concepts, entities, or approaches discussed.

Table structure:
- Each column represents one concept/entity/approach to compare
- Each row represents an evaluation dimension or criterion

Suggested dimensions (adapt to the content):
- Definition / Core purpose
- Key strengths
- Key weaknesses / limitations  
- When to use / ideal context
- Evidence quality in sources
- Overall rating (1-5)

Instructions:
- Use concise, parallel language across all cells
- Bold the winner/best option in each row where applicable
- Add a "Summary" row at the bottom with overall recommendation
- Add footnotes for any controversial or uncertain comparisons
```
