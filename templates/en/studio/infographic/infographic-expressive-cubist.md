# Infographic — Expressive Cubist Abstract

**Source:** Custom - Original
**Orientation:** Square (1:1)
**Detail Level:** Detailed

## Steering Prompt

```
# infographic_design_spec_expressive_cubist.yaml
# Style: Expressive Abstract Cubist
# Concept: "Fractured Reality, Vivid Emotion"

Global Design Settings:
  Tone: "Energetic, emotive, contemporary fine art, gallery aesthetic"

  Color Palette:
    Primary:
      - "#FF1493 (magenta — vibrant emotional anchor)"
      - "#00CCCC (cyan — cool counterpoint)"
      - "#7B2FBE (violet — deep expressive accent)"
    Secondary:
      - "#FF6B4A (coral — warm tension)"
      - "#00C8B0 (turquoise — harmonic balance)"
    Rule: "High-saturation palette. Warm and cool tones in direct contrast. No muted or neutral zones."

  Surface:
    Texture: "Visible canvas texture — rough brush strokes, layered transparency, paint splashes"
    Medium: "Watercolor and acrylic mixed-media effect. Color bleeding at edges."
    Finish: "Hand-painted textured surface, gallery-style modern art finish"

  Typography:
    Headings: "Bold, fragmented or slightly distorted letterforms. Strong visual weight."
    Body: "Clean sans serif for data — contrast against painterly background."
    Numbers: "Oversized, integrated into geometric planes as visual elements."
    Language: "The language should match the user's prompt language."

  Composition:
    Structure: "Asymmetrical — dynamic angular planes overlapping. No rigid grid."
    Fragmentation: "Cubist-inspired deconstruction — subjects broken into geometric shards."
    Depth: "Soft blended gradients behind sharp broken edges. Painterly distortion."
    Contrast: "Strong contrast between warm and cool tones throughout."

Layout Variations:
  - Type: "Geometric Fragment Hero"
    Design: "Central subject deconstructed into overlapping angular planes. Data points embedded within geometric shards. Each fragment contains a single fact or figure. Background: expressive paint splashes and color bleeds."

  - Type: "Cubist Data Grid"
    Design: "Information divided into irregular polygonal cells, each a different saturated color block. Data point centered in each cell with contrasting typography. Cells overlap slightly at edges with transparency."

  - Type: "Abstract Timeline"
    Design: "Left-to-right progression of geometric events. Each milestone: a fragmented angular shape in a different palette color. Bold numbers as oversized visual anchors. Rough brush stroke connectors between elements."

  - Type: "Emotional Contrast Split"
    Design: "Composition divided diagonally. Warm tones (magenta/coral) on one half, cool tones (cyan/violet) on the other. Data presented symmetrically on both sides to invite comparison."

Design Rules:
  - "Fragmentation First: Every element — text, data, imagery — is embedded in or adjacent to a geometric shard."
  - "Color Tension: Warm and cool palette must both appear on every layout. No monochromatic zones."
  - "Texture Layer: Canvas grain and brush strokes are always visible — never flat digital fills."
  - "Asymmetry: Perfectly centred or grid-aligned compositions are forbidden. Embrace controlled chaos."
  - "Data as Art: Statistics and labels are typographic graphic elements, not secondary annotations."
```
