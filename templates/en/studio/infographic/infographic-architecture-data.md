# Infographic — Architecture/Data

**Source:** Custom - Validated
**Orientation:** Landscape (16:9)
**Detail Level:** Standard

## Steering Prompt

```
Isometric technical diagram, clean blueprint aesthetic, soft blue and grey palette with teal accents.
Show components as connected nodes with directional data flow arrows.
Use minimal text labels, rely on icons and visual hierarchy.
Structure as left-to-right flow: sources → processing → storage → consumption.
```

## Variante Concise

```
Isometric diagram, blueprint style, blue-grey duotone.
Components as nodes, arrows showing data flow direction.
Minimal text, icons only.
```

## Use Case (Data Engineering)

Visualiser architecture data platform pour documentation technique. Idéal pour :
- Diagrammes Medallion Architecture (Bronze/Silver/Gold)
- Pipelines ETL/ELT avec flux directionnels
- Architectures cloud (Azure Databricks, Snowflake, etc.)

**Test validé avec** : Microsoft Learn - Azure Databricks Architecture
