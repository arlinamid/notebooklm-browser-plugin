# Infographic — Hybrid Conceptual Collage

**Source:** Custom - Original
**Orientation:** Portrait (2:3)
**Detail Level:** Detailed

## Steering Prompt

```
# infographic_design_spec_hybrid_conceptual_collage.yaml
# Style: Hybrid Conceptual Collage
# Concept: "Portrait of Information — Architecture of Thought"

Global Design Settings:
  Tone: "Modern intellectual, cinematic, experimental mixed-media, editorial poster"

  Color Palette:
    Base: "#EDE8DC (muted beige — worn paper foundation)"
    Primary:
      - "#1A1A1A (charcoal black — dominant structure)"
      - "#6B6B6B (muted gray — grayscale mid-tones)"
    Accent:
      - "#E85D04 (bold orange — focal anchor and highlights)"
    Rule: "Grayscale dominates 80%. Bold orange on highest-priority elements only — maximum 20% surface."

  Surface:
    Background: "Distressed grunge — subtle cracks, stains, worn paper texture"
    Overlay: "Soft smoky cloud overlays blending composition layers seamlessly"
    Texture: "Layered cut-and-paste paper texture. Semi-matte finish."

  Typography:
    Headings: "Strong geometric or grotesque sans serif. High contrast weight."
    Body: "Clean condensed type. Blueprint or technical label aesthetic."
    Technical: "Blueprint grid lines and measurement annotations as decorative elements."
    Language: "The language should match the user's prompt language."

  Composition:
    Layers:
      - "Back: Distressed beige paper texture with subtle stains"
      - "Mid-back: Grayscale portrait or architectural rendering (double exposure)"
      - "Mid: Geometric grid overlay, blueprint technical lines, fragmented square blocks"
      - "Front: Bold orange accent elements and primary data typography"
    DoubleExposure: "Realistic grayscale portrait rendering blended with urban architectural montage"
    Pixelation: "Fragmented square blocks and pixelated mosaic elements at composition edges"
    NegativeSpace: "Balanced negative space — bold orange elements surrounded by breathing room"
    Lighting: "Cinematic — strong contrast, soft smoky transitions between layers"

Layout Variations:
  - Type: "Editorial Poster"
    Design: "Full-height grayscale portrait double-exposed with architectural skyline. Orange horizontal rule divides title from body. Blueprint grid lines overlay the portrait. Geometric pixel fragments at top and bottom edges."

  - Type: "Blueprint Data"
    Design: "Technical grid as primary structure. Data in blueprint-style cells with measurement lines. Grayscale portrait ghosted behind the grid at reduced opacity. Orange highlights on key figures."

  - Type: "Fragmented Identity"
    Design: "Portrait split into mosaic of different-sized square blocks, each slightly offset. Some blocks contain data, some complete the portrait, some are replaced with architectural fragments. Orange accent block as focal anchor."

  - Type: "Urban Archive"
    Design: "Left half: architectural collage montage in grayscale. Right half: clean data panel on beige. Orange dividing rule between halves. Blueprint annotations overlaying the architectural elements."

Design Rules:
  - "Double Exposure: Every layout must blend portrait/figure with architectural or structural elements."
  - "Orange Discipline: Bold orange on maximum 2 elements per layout — title and one accent."
  - "Texture Depth: Three distinct texture layers minimum (paper, grunge, smoky overlay)."
  - "Blueprint Lines: Technical grid or measurement lines must appear on every layout."
  - "Cinematic Contrast: Strong lighting contrast throughout. Deep shadows, sharp highlights."
```

## Compact Variant

```
Hybrid conceptual collage illustration, double exposure composition, editorial poster aesthetic, fusion of realistic grayscale portrait rendering with urban architectural montage, geometric grid overlays and technical blueprint lines, fragmented square blocks and pixelated mosaic elements, layered cut-and-paste paper texture, distressed grunge background with subtle cracks and stains, bold orange and charcoal black color accents over muted beige base, soft smoky cloud overlays blending elements seamlessly, cinematic lighting with strong contrast, semi-matte texture, modern intellectual vibe, experimental mixed-media digital art, high detail, sharp focus, balanced negative space.
```

## Use Case

Intellectually sophisticated editorial infographic for complex subjects. Ideal for:
- Biographical, sociological, or urban studies content
- Architecture, technology, and innovation topics
- Editorial journalism, think-piece illustration, academic posters
- Audiences who value layered conceptual visual storytelling

**Orientation:** Portrait — editorial poster format reads top to bottom, double exposure works best vertically.
**Detail Level:** Detailed — multiple layered composition elements need comprehensive guidance.
