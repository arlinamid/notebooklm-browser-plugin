# Infographic — Brutalist Editorial

**Source:** Custom - Original
**Orientation:** Portrait (2:3)
**Detail Level:** Standard

## Steering Prompt

```
# infographic_design_spec_brutalist_editorial.yaml
# Style: Ultra-Modern Brutalist Editorial
# Concept: "Controlled Anarchy in Print"

Global Design Settings:
  Tone: "Avant-garde, urban underground, premium poster, experimental, chaotic yet structured"

  Color Palette:
    Primary:
      - "#000000 (pure black — dominant structure, typography, graphic elements)"
      - "#FFFFFF (pure white — negative space, light zones, contrast panels)"
    Accent:
      - "#FF0000 (pure red — single bold accent, maximum visual impact)"
    Rule: "Strictly three colors only: black, white, red. No gradients. No additional hues."

  Surface:
    Texture: "Worn ink textures, grainy xerox noise, stencil spray marks"
    Collage: "Layered collage — torn paper edges, overlapping text blocks, truncated titles"
    Finish: "Distressed grunge. Premium poster print quality meets underground aesthetic."

  Typography:
    Primary: "Swiss International Typographic Style — bold grotesque, precise grid alignment"
    Experimental: "Fragmented titles, truncated words, text running off the edge, rotated labels"
    Stencil: "Stencil-style uppercase labels for section markers and callouts"
    Overlap: "Overlapping text blocks where type layers on type — controlled visual tension"
    Language: "The language should match the user's prompt language."

  Composition:
    Grid: "Asymmetric grid system — Swiss grid broken and reassembled expressively"
    Structure: "Fragmented geometric shapes, abstract architectural lines as organizational tools"
    Hierarchy: "Magazine cover aesthetic — one dominant element, everything else subordinate"
    NegativeSpace: "Large black or white zones as deliberate breathing space and visual anchors"

Layout Variations:
  - Type: "Brutalist Magazine Cover"
    Design: "Large black headline bleeds off the top edge, partially truncated. Red block accent cuts across mid-section. White negative space dominates lower third with minimal data. Xerox grain throughout."

  - Type: "Swiss Grid Deconstructed"
    Design: "Strict column grid structure — then deliberately broken. Some columns shifted, some text rotated 90°. Red accent appears in exactly one column. Black and white alternating column backgrounds."

  - Type: "Collage Archive"
    Design: "Layered torn paper elements at different scales. Text runs across boundaries between layers. Black stencil labels over white zones. Red marker line connecting disparate data points."

  - Type: "Underground Poster"
    Design: "Full black background. White and red elements only. Bold title in massive white type, partially hidden behind a black geometric block. Data as small tight type clusters. Red accent line as the sole color element."

Design Rules:
  - "Three Colors Only: Black, white, red — no exceptions, no gradients, no tints."
  - "Typography as Architecture: Type is a structural element, not just a label. It builds the composition."
  - "Controlled Chaos: Every broken element is deliberately placed — chaos is designed, not accidental."
  - "Xerox Texture: Grainy noise and worn ink must appear on every layout — no clean digital surfaces."
  - "Truncation Allowed: Titles and headings may bleed off the edge intentionally. Incompleteness is a design choice."
  - "Red Economy: Red appears on maximum one element per layout — its scarcity is its power."
```
