# Infographic — Process/Timeline

**Source:** Custom - Validated
**Orientation:** Portrait (9:16)
**Detail Level:** Standard

## Steering Prompt

```
Isometric journey illustration, winding path flowing top-to-bottom.
Soft blue and teal palette with milestone markers in warm accent colors.
Each step as a distinct station along the path with numbered icons.
Clean connecting lines showing progression and dependencies.
Minimal text labels, visual storytelling through icons and spatial flow.
```

## Variante Timeline Classique

```
Vertical timeline infographic, clean modern aesthetic.
Central spine with alternating left-right content blocks.
Numbered milestones with distinct icons per stage.
Soft gradient background, professional color palette.
Clear visual hierarchy: stage number → title → brief description.
```

## Use Case (Data Engineering)

Documenter des processus séquentiels. Idéal pour :
- Pipelines CI/CD (7 étapes)
- Workflows de déploiement
- Processus d'onboarding technique
- Data lifecycle management

**Test validé avec** : Tutorial Works - 7 Essential CI/CD Pipeline Stages

**Note** : Meilleur résultat de la série. La combinaison "isometric journey + winding path + numbered stations" produit une métaphore visuelle très efficace (chemin sinueux Z-flow).
