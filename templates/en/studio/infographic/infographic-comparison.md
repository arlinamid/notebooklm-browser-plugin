# Infographic — Comparaison

**Source:** Custom - Validated
**Orientation:** Landscape (16:9)
**Detail Level:** Standard

## Steering Prompt

```
Split-screen comparison layout, high-contrast duotone palette.
Left side: cool blue tones. Right side: warm orange tones.
Central vertical axis listing comparison criteria.
Clean grid alignment, sans-serif typography, minimal decoration.
Use icons to represent features, checkmarks for capabilities.
```

## Variante Swiss

```
Side-by-side comparison infographic, Swiss design aesthetic.
Two distinct color zones separated by central divider.
Structured rows for each comparison criterion.
Icons and visual indicators over text descriptions.
Professional monochrome with accent colors for differentiation.
```

## Use Case (Data Engineering)

Comparer deux approches ou technologies. Idéal pour :
- ETL vs ELT (avec critères alignés)
- Batch vs Streaming processing
- Comparaison d'outils (dbt vs Dataform, Airbyte vs Fivetran)

**Test validé avec** : AWS - ETL vs ELT

**Note** : La directive "split-screen" produit une structure 4 colonnes très lisible avec différenciation couleur parfaite.
