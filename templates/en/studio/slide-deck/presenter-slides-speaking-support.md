# Presenter Slides - Speaking Support

**Source:** vibeproductmarketing  
**Type:** Presenter Slides

## Prompt

```
Create Presenter Slides for [AUDIENCE: team standup / knowledge sharing].

Structure:
- Title slide with key message
- 3-5 content slides (one idea per slide)
- Visual emphasis on diagrams/charts
- Speaker notes with talking points
- Closing slide with next steps/actions

Style: Clean, visual, minimal text (bullet points only for speaker reference)
Include: Prompts for discussion, questions to ask audience
Exclude: Full sentences on slides, dense information

Length: Short
Output language: [French/English]
```

## Use Case (Data Engineering)

Présentation rapide pour partage de connaissances en équipe.
