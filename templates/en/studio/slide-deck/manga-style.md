# Manga Style

**Source:** Original Library
**Format:** Slide Deck
**Difficulty:** Beginner

## Prompt

```
# presentation_design_spec_manga_comic.yaml
# Style: Manga / Comic Book
# Concept: "Story as Structure — Information through Sequential Art"

Global Design Settings:
  Tone: "Dynamic, expressive, narrative-driven, energetic yet readable"

  Color Palette:
    Base: "#FFFFFF (clean white — comic paper)"
    Text: "#000000 (strong ink black)"
    Accent:
      - "#FF4444 (impact red — speed lines, emphasis, danger)"
      - "#FFD700 (highlight yellow — sparkles, focus moments)"
    Tones: "Halftone dot patterns and screen tones for shading (gray areas)"
    ColorRule: "Selective spot color: maximum 1–2 accent colors per slide. Default: black, white, one accent."

  Typography:
    Headings: "Bold hand-lettered or manga-style typeface. All caps or mixed caps for impact."
    Body: "Comic-style or clean rounded sans serif inside speech bubbles."
    SFX: "Large sound effect typography (POW!, ZOOM!, HA!) used as graphic elements."
    Numbers: "Bold, oversized — treated as visual elements, not mere labels."
    Language: "The language should be what users said in the prompt."

  Common Layout Rules:
    Panels: "Divide slides into 2–4 manga panels separated by thick black gutters (4–6pt borders)."
    SpeechBubbles: "Use speech bubbles and thought clouds for key information, quotes, and data points."
    ReadingFlow: "Left-to-right, top-to-bottom panel sequence (Western manga convention)."
    SpeedLines: "Add radial speed lines behind key elements to create visual impact and focus."
    ScreenTones: "Use halftone dot patterns for shadow areas — not solid gray fills."

Layout Variations:
  - Type: "Title / Cover Page"
    Design: "Full-bleed composition. Central character or concept illustration dominates. Large manga-style title lettering with dramatic speed lines radiating outward. Small subtitle in a clean rectangular caption box at the bottom."

  - Type: "Four-Panel Story Grid"
    Design: "Classic 2×2 panel grid. Each panel shows one step, concept, or data point. Panels vary in size to show importance — the most critical panel is largest. Thick black borders (4–6pt) between panels."

  - Type: "Speech Bubble Data"
    Design: "Multiple overlapping speech bubbles of varying sizes, each containing a key fact, quote, or statistic. Characters or illustrated personas speak the data. Background is a single flat color or screen tone."

  - Type: "Big Impact Spread"
    Design: "Single full-slide panel with a dramatic scene. One massive number or phrase centered. Speed lines radiate from it. A small caption box in the corner explains context."

  - Type: "Chibi Explainer"
    Design: "A small deformed (chibi) character guides the viewer through a step-by-step process. Each step labeled in a rectangular caption box. Character reacts expressively (surprised, determined, celebratory) at each point."

  - Type: "VS Battle Layout"
    Design: "Screen split diagonally. Left: Option A with its character or icon. Right: Option B. A lightning bolt or jagged line divides them. Pros and cons listed in caption boxes on each side."

  - Type: "Timeline / Chapter Flow"
    Design: "Horizontal panel strip across the slide. Each panel = one time period or phase. Connected by directional arrows in manga speed-line style. Key moments shown as character reaction panels."

Design Rules:
  - "Outlines: All illustration elements use thick, confident black ink outlines (3–5pt minimum). No thin hairlines."
  - "Screen Tones: Use halftone dot patterns for shadow areas — never solid gray fills."
  - "Panel Borders: Gutters between panels are always solid black (4–6pt). Panels define the structure."
  - "Expressions: Characters must have clear, exaggerated emotional reactions — no neutral faces."
  - "Color Restraint: Maximum 2 spot accent colors per slide. Default is black, white, and one color."
  - "No Corporate Templates: No gradient backgrounds, no drop shadows, no PowerPoint default styles."
  - "Generation: Generate a cohesive multi-slide presentation where each slide functions as a manga page. Maintain consistent character design, panel border style, and typographic voice throughout."
```
