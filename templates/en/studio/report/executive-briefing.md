## Prompt

```text
Create a formal executive briefing report based on the source material.

Structure:
1. **Executive Summary** (3-4 bullets, top findings only)
2. **Context & Background** (What problem or situation this addresses)
3. **Key Findings** (5-7 most significant insights, with source citations)
4. **Analysis** (What the data means, patterns, implications)
5. **Strategic Recommendations** (3-5 actionable items with priority level)
6. **Appendix** (Supporting data, methodology notes)

Requirements:
- Tone: Professional, objective, incisive
- Length: Comprehensive but scannable (use headers, bullets, bold key terms)
- Audience: Senior leadership / decision-makers
- Every claim must reference the source material
- Conclude with clear next steps
```
