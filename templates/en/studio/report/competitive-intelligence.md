## Prompt

```text
Create a formal competitive intelligence report based on the source material.

Include these sections:
1. **Market Overview**: Current state, size, key trends
2. **Competitive Landscape**: Main players, their positioning, strengths/weaknesses
3. **Feature/Capability Comparison**: Side-by-side analysis table of key dimensions
4. **Strategic Gaps & Opportunities**: Where gaps exist in the market
5. **Threat Assessment**: Ranking of competitive threats (High/Medium/Low)
6. **Recommended Positioning**: How to differentiate based on the evidence

Tone: Analytical, data-driven, strategic
Format: Use tables where comparative data exists; use bullet points for qualitative insights.
Include confidence levels (High/Medium/Low) for each major claim based on source strength.
```
