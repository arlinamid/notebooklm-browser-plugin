# Seminar Use (Minimal Text)

**Source:** Kawai
**Format:** Slide Deck
**Difficulty:** Intermediate

## Prompt

```
# presentation_design_spec_seminar_minimal.yaml
# Style: Seminar Minimal Text
# Concept: "One Idea, Total Presence"

Global Design Settings:
  Tone: "High-sensibility, refined, sophisticated, presenter-focused, minimalist editorial"

  Color Palette:
    Base: "#FFFFFF (pure white — primary canvas)"
    Text: "#111111 (near-black — precise, sharp)"
    Accent:
      - "#D32F2F (signal red — single focal emphasis only)"
    Secondary:
      - "#F5F5F5 (light gray — subtle section backgrounds)"
      - "#BDBDBD (medium gray — secondary text, dividers)"
    Rule: "Red appears on maximum one element per slide — never decorative, always purposeful."

  Typography:
    Headings: "Clean bold sans-serif with generous letter spacing. Large scale — occupies 40–60% of slide height."
    Body: "Thin-weight sans serif. Very small. High contrast against white background."
    Accent: "Red used on a single word, number, or short phrase — never more than one per slide."
    Language: "The language should be what users said in the prompt."

  Common Layout Rules:
    TextDensity: "Maximum 15 words per slide. Slides are visual anchors for the speaker, not self-contained documents."
    Photography: "One high-quality editorial-quality photo per slide where used. Never stock photo feel — fashion portrait or still-life quality."
    Alignment: "Asymmetric — titles flush left or anchored to a grid, never centered unless for deliberate dramatic effect."
    WhiteSpace: "Minimum 40% of each slide must be empty white space."

Layout Variations:
  - Type: "Opening Statement"
    Design: "All-white slide. One short, powerful phrase (4–7 words) in massive bold type, lower-left anchored. Red underline beneath one key word. Nothing else."

  - Type: "Photo + Headline"
    Design: "50:50 split. Left: a single editorial-quality portrait photo (desaturated or cool-toned). Right: one bold headline in large type with optional one-line body text below. Red accent on one word."

  - Type: "Big Number"
    Design: "White canvas. One enormous statistic at near-full-height in near-black. Unit or brief label in small body text to the right. One red element for emphasis. Maximum 8 words total."

  - Type: "Minimal List"
    Design: "3–5 single-word or two-word items arranged vertically. Each item on its own line. Large, bold, generous spacing. One item highlighted in red. No bullets, no icons."

  - Type: "Full Photo"
    Design: "Full-bleed editorial photo. One 2–4 word headline overlaid in white or black, bottom-left placement. A thin red horizontal rule above the text. Zero other elements."

  - Type: "Definition Slide"
    Design: "White background. One term or concept in massive bold type (left-aligned, top third). Below it: one clean, concise definition sentence in small body text. Red colon or dash as separator."

  - Type: "Closing Anchor"
    Design: "White slide. One final statement or call-to-action in very large bold type. Red period or punctuation mark as the only accent. No logo, no decoration — pure statement."

Design Rules:
  - "Word Economy: Every word on screen earns its place. Delete anything that can be said verbally by the presenter."
  - "Red Restraint: One red element maximum per slide. It highlights the single most important thing."
  - "Photo Quality: Only use photography that feels art-directed — natural light, editorial composition, never clipart."
  - "No Bullet Points: Information is presented as statements, not lists (unless using the Minimal List layout)."
  - "No Decorative Elements: No icons, no infographic shapes, no ornaments — unless they ARE the content."
  - "Generation: Generate slides that function as speaker support, not self-contained documents. Each slide communicates one idea with maximum visual economy."
```
